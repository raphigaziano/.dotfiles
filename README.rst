-1 Clone this repo in $HOME:

   :

       ~ $ git clone https://github.com/raphigaziano/.dotfiles.git

-2 (Opt) Install packages:

   :

      ~ $ .dotfiles/scripts/getpkgs.sh      # From package manager
      ~ $ .dotfiles/scripts/pypkgs.sh       # Python packages
      
-3 Update config:

   :

      ~ $ .dotfiles/install.sh

-4 profit!
